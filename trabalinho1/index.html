• para acessar a pasta do repositorio

    cd < nome da pasta >

  • setar usuário

    git config--global user.name" nome "

  • setar email

    git config --global.email" email "

  • criar novo repositório

    git init

  • verificar estado dos arquivos/diretórios

    git status

  • adicionar todos os arquivos/diretorio

    git add

  • comitar um arquivo

    git commit -m" texto falando dia da atualização "

  • informar a pasta remota.

    git remote add origin http:// < link >

  • publica as alteraçães no repositorio remoto.

    git push -u origin master

  • criando novo branch

    git branch bug-123

  • criar um novo branch e trocar

    git checkut -b bug-456

  • baixar as alterações no repositório remoto.

    git pull
